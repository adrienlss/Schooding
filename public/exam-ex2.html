Décaler les valeurs d'une matrice

Soit une matrice d'entiers en entrée et une valeur entière qui représente un décalage.

Il vous est demandé que la matrice soit modifiée de telle manière que les valeurs de cette matrice soient décalées de manière circulaire.

Vous aurez soin de tester les cas d'une matrice null ou de longueur 0 auquel cas la matrice est inchangée.

static void DecaleMatrice(int[,] mat, int valeur)

Exemple : soit la matrice suivante {{1,2},{3,4}}

avec un décalage de 1, on obtient alors {{4,1},{2,3}}

avec un décalage de 2, on obtient alors {{3,4},{1,2}}
